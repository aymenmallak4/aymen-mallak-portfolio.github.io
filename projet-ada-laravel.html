Projet ADA — Site internet Laravel (HTML, PHP, MySQL)

Application frontend/backend permettant à tout internaute de consulter les données de ADA : agences, véhicules, offres de location, réservations.

Date limite : 19/12/2025 BTS SIO SLAM — 2ᵉ année Projet académique
Contexte du projet

Phase 2 du projet ADA, réalisée en groupe de deux en mode agile Scrum. L'objectif est de développer l'application frontend/backend permettant à tout internaute de consulter les données de ADA (agences, véhicules, offres de location).

L'application accède à une base de données MySQL centralisée, commune à toutes les agences. Outils de gestion de projet : Trello / Projeqtor, et éventuellement Github.

Normes de développement
  • Développement en PHP avec le Framework Laravel (incluant l'ORM Eloquent)
  • Architecture MVC (Modèle-Vue-Contrôleur)
  • Mise en forme frontend avec Bootstrap (inclus dans Laravel)
  • Code indenté et commenté
Base de données

La structure et les données de la base existante SqlServer (Phase 1) ont été migrées vers MySQL. La norme de nommage des tables et champs pour Eloquent a été appliquée. Le script SQL est fourni en pièce jointe.

Fonctionnalités — User Stories Scrum
1 Informations d'une agence

Sélectionner une agence dans une liste déroulante pour visualiser ses informations complètes : adresse, téléphone, email, horaires d'ouverture, et tous ses véhicules (marque, modèle, libellé et photo catégorie).

2 Offre de location ADA

Sélectionner un genre (liste déroulante ou icônes cliquables), puis une catégorie correspondante, pour visualiser ses informations complètes : libellé, photo, attributs (logo, libellé, valeur).

3 Création d'un compte client

En tant que visiteur, se créer un compte client.

4 Connexion / déconnexion

Se connecter avec son compte client et se déconnecter. Le menu s'adapte selon la situation (connecté ou non).

5 Réservation d'une location

En tant que client connecté : saisir une date de début et de fin, sélectionner une agence, visualiser les véhicules disponibles (non déjà réservés sur la même période), puis sélectionner un véhicule pour enregistrer la réservation.

6 Mes contrats de location

En tant que client connecté : visualiser tous ses contrats de location (passés ou en cours), puis cliquer sur un contrat pour en voir les détails : dates, nbkm, montant, agence, véhicule.

Technologies utilisées
Laravel PHP MySQL Eloquent ORM MVC Bootstrap HTML
