Projet ADA — Application Visual Studio C# Entity

Application back-office permettant au responsable d'une agence d'administrer les données de son agence de location de véhicules.

Date limite : 5/11/2025 BTS SIO SLAM — 2ᵉ année Projet académique
Contexte du projet

Projet réalisé en équipe Scrum (binôme) dans le cadre du BTS SIO SLAM 2ᵉ année. L'objectif est de développer l'application back-office permettant au responsable d'une agence d'administrer les données de son agence de location de véhicules.

L'application est déployée dans chaque agence avec un accès à une base de données SqlServer centralisée, commune à toutes les agences. Outils de gestion de projet : Trello / Projeqtor, et éventuellement Github.

Conception de la base de données

La base de données est conçue à partir du diagramme de classes fourni. Un script partiel est disponible ; la structure est complétée en créant les tables manquantes et les relations entre clés étrangères et clés primaires. Les données de test sont insérées manuellement.

Normes de développement
  • Programmation orientée objet avec Microsoft Visual Studio C# et le Framework Entity
  • Architecture MVC
  • Règles de nommage, commentaires, indentation
  • Tests et intégration en continu
Fonctionnalités — User Stories SCRUM
1 Connexion

Saisir l'identifiant et le mot de passe de l'agence sur le formulaire de connexion pour accéder à l'écran d'accueil.

2 Modification des informations de l'agence

Modifier le nom, l'adresse, le téléphone de l'agence, et les heures d'ouverture pour chacun des 7 jours de la semaine.

3 Gestion du parc de véhicules

Visualiser tous les véhicules de l'agence, filtrer par catégorie. En cliquant sur un véhicule, afficher la liste des attributs de sa catégorie.

4 Ajout / suppression d'un véhicule

Ajouter un nouveau véhicule ou supprimer un véhicule existant.

5 Gestion des fiches clients

Créer une nouvelle fiche client ou modifier la fiche (rue, cp, ville, téléphone, email) d'un client existant via son n° identifiant.

6 Visualisation des contrats de location

Visualiser tous les contrats de location de l'agence avec tous les détails, avec filtre par date de début ou date de fin.

7 Ajout d'un contrat de location Annulé

Ajouter un nouveau contrat de location (datesignature, dateheuredebut, dateheurefin) pour un client sélectionné dans une liste.

8 Modification d'un contrat Annulé

Modifier un contrat lors du début (nbkmdebut) ou de la fin (nbkmfin, montantfacture) d'une location via le n° identifiant du contrat.

Technologies utilisées
C# Visual Studio 2022 Entity Framework SQL Server POO / MVC

Bilan

Application back-office de gestion d'agence de location de véhicules, développée en binôme en mode agile SCRUM avec Visual Studio C# Entity Framework et SQL Server. 6 User Stories réalisées sur 8 (US 7 et 8 annulées).
